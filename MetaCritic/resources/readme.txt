Metacritic - XBMC Programs/Plugins to display metacritic score / reviews


SETUP:
 Install to Plugins\Programs\Metacritic  (IMPORTANT! keep sub folder structure intact)


USAGE: 
 1) From XBMC Home Screen choose "Programs"   (you may need to first enable this in your chosen XBMC skin)
 2) Select "Program Plugins"
 3) Select "Metacritic"
    Plugin will run and present you with is Root Categories.
 4) Select a category and a list of items will be shown
 5) Select an item to display item details
